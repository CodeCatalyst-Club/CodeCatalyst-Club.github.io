RULES AND REGULATIONS

  1. Familiarize yourself with the competition rules and guidelines before beginning to code.
  2. Choose a quiet and distraction-free environment to work in.
  3. Make sure you have a reliable internet connection and a charged laptop or desktop computer.
  4. Do not cheat in any way during the competition. This includes copying code from other sources, using unauthorized resources, or collaborating with others outside of the permitted methods.
  5. Test your code frequently to ensure it is functioning correctly.
  6. Optimize your code for performance where possible, but avoid over-optimization at the expense of readability.
  7. Submit your code before the deadline and make sure to check for any errors or omissions before submitting
  8. Do not change tab while attending the challenge
  9. Keep a clear and organized coding style. Use indentation, spacing, and commenting to make your code more readable and easier to follow.
